# Porsche Build Experience

This is a lil interactive frontend project where users can explore different Porsche build styles.

## Overview

Users can select between RWB, Singer, and Gunther Werks builds. Based on the selection, the page dynamically updates the visuals, branding, and content to match the chosen style.

## Features

- Interactive buttons to switch between builds
- Dynamic content updates using JS
- Background and UI changes based on selection
- External brand links for each build
- Embedded social media content for context

## Built With

- HTML
- CSS
- JavaScript

## Purpose

This project was created to practice frontend development, focusing on UI interaction, layout, and creating a more engaging user experience.
